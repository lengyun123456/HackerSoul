我们认为Flat-UI拥有一套非常优美的UI，遂翻译成中文文档供更多的人使用。

Flat UI Free 是一套精美的扁平风格 UI 工具包，基于 Twitter Bootstrap 实现。
这套界面工具包包含许多基本的和复杂的 UI 部件，例如按钮，输入框，组合按钮，复选框，单选按钮，标签，菜单，进度条和滑块，导航元素等等。
